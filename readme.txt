Description:

This is the KNN Data Mining Implementation Algorithm. This program will generate a KNN model according to the input training data. Also, it will classify the input test data. The input data should be in ".arff" format.

User Guide: 
To run the program, simply run the main.java in the Main package. See detailS in the comment.

Files end with "Selection" are for Part A
Files end with "Intro" are for Part B
